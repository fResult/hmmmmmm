# การติดตั้ง
เข้าไปในโฟลเดอร์ `hmmmmmm`
```bash
cd hmmmmmm
```

รันคำสั่งติดตั้ง Package
```base
npm install
# or
yarn
# or
pnpm install
```

รอจนกว่าการติดตั้งจำเสร็จสิ้น  
แล้วสั่ง run ให้ app นี้ทำงาน
```
npm run dev
# or
yarn dev
# or
pnpm run dev
```

จากนั้นเปิด Browser เข้า url `http://localhost:3000`
